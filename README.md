Веб-приложение — визуальная утилита для анализа финансовых графиков.  

## Функциональность

- Отображение свечного графика (OHLC)
- Выбор валютной пары через API-запрос
- Поддержка таймфреймов:
  - 1M, 5M, 15M, 1H, 4H, 1D
- Переключение темы (light / dark)
- Сайдбар с индикаторами:
  - MACD
  - RSI
  - MFI
- Инструменты рисования на графике:
  - линии
  - прямоугольники
- Синхронизация нескольких графиков

## Технологии

- Next.js
- React
- TypeScript
- lightweight-charts
- Zustand
- Tailwind CSS
- shadcn/ui
- next-themes
- lucide-react
